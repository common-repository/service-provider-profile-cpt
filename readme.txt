Creates “Services”, “Provider” custom post type. 

== Description ==

Creates “Provider” custom post type. Additional meta boxes will be available in "Add New People" section - job title, phone number, email address, staff booking page URL, linkedin, twitter, facebook, google+.

Creates “Services” custom post type. Additional meta boxes will be available in "Add New Staff” section - service cost, service duration.

== Installation ==

Please refer to [online documentation page](http://support.setmore.com/article/553-setmore-theme-custom-post-type) for more details.

1. Download the plugin.
2. Extract the folder. 
3. Upload the Service Provider Profile CPT SetMore plugin folder to the '/wp-content/plugins/' directory.
4. Activate the plugin through the 'Plugins' menu in WordPress. An item called “Staff”, “Service” will be added to the left menu.
